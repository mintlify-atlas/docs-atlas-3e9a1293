RemoteTerm is a self-hosted backend server and browser interface for MeshCore mesh radio networks. Connect your radio via USB serial, TCP, or Bluetooth, then access messaging, packet analysis, automation bots, and network visualization from any browser on your network.

## Get started

<Steps>
  <Step title="Install RemoteTerm">
    Clone the repository and install dependencies using `uv` and Node.js, or pull the Docker image.
  </Step>
  <Step title="Connect your radio">
    Set your transport environment variable — `MESHCORE_SERIAL_PORT`, `MESHCORE_TCP_HOST`, or `MESHCORE_BLE_ADDRESS` — and launch the server.
  </Step>
  <Step title="Open the browser interface">
    Navigate to `http://localhost:8000` to access the full messaging and management UI.
  </Step>
  <Step title="Configure integrations">
    Add MQTT forwarding, Python bots, webhooks, or Home Assistant discovery from the Settings panel.
  </Step>
</Steps>

<Warning>
  RemoteTerm is designed for trusted environments only. Do not expose it to an untrusted network without enabling HTTP Basic Auth and HTTPS. The bot system executes arbitrary Python code — anyone with access to the UI can run code on your server.
</Warning>
